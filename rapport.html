CU: Créer une séance

Objectif: Un professionnel souhaite donner une séance d'un service au GYM

Acteurs
  • Professionnel (Principal)
  • Agent (Secondaire)
Préconditions
  • Le professionnel a un compte et est présent au GYM.
Postconditions
  • Aucun
Notes

Notes additionnelles et hypothèses

Scénario principal
  1. Le professionnel se présente à l’accueil et donne son numéro.
  2. L’agent entre le numéro dans le système.
  3. Le système valide le numéro du membre.
  4. Le professionnel indique à l’agent le service qu’il souhaite fournir.
  5. L’agent consulte le Répertoire des services pour trouver le code de service.
  6. L’agent demande au professionnel le reste des informations requises.
  7. L’agent entre les données dans le système.
  8. Le système affiche les détails de la séance nouvellement créée.
Scénarios alternatifs
  1. Le compte est suspendu.
    1. Le système affiche Compte suspendu à l'écran
    2. Le scénario se termine
  1. Le service souhaité n'est pas offert chez GYM.
    1. L'agent ne trouve pas le code pour le service demandé.
    2. L'agent informe le professionnel de la non-disponibilité du service.
    3. Le scénario se termine.
